# 信長の野望・ミニマム (MVP)

「信長の野望」の核心的なゲーム体験（領地の拡大と支配）を極限まで抽出した、Pythonによるミニマルなプロトタイプです。

## 概要
このプログラムは、コマンドライン上で動作するシミュレーションです。プレイヤーは強力な武将「織田」として、周辺の城を攻略し、天下統一を目指します。

### 実装されている機能
- **ターン制システム**: プレイヤーとAI（他勢力）が交互に行動します。
- **領土奪取**: 隣接する拠点へ軍を進軍させ、勝てばその拠点の所有権を獲得。
- **兵数による判定**: 進軍時の勝利は単純に「自軍の兵数」と「敵の兵数」の比較で行われます。

## セットアップと実行方法

### 必要な環境
- Python 3.x 以上

### 実行手順
1. このディレクトリにある `main.py` を開く、あるいはパスを通した状態で以下を実行します。
```bash
python main.py
```

2. 表示される現在の状況を確認します。
   - 各拠点の名前、所有者（Ooda, Takeda, Hideyoshiなど）、残りの兵数が表示されます。

3. 命令を入力します。
   - 進軍する場合: `move X Y` (例: `move 1 0`) を入力します。
     - ※XとYはマップ上の座標です。
   - 何もしない場合: `pass` と入力してください。

## ゲームのルール
- あなた（Oda）の目的は、すべての拠点の所有者を「Oda」にすることです。
- 進軍先の座標を入力すると、隣接するあなたの拠点から兵が移動し、戦闘が行われます。
- 勝利すればその地を奪取でき、兵数も蓄積されます。
